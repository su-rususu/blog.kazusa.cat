Redmi Note 7 Pro を買いました。

9月 26 2019

Huawei nova liteに限界を感じたので。2019年大失敗グランプリはMoto G5S Plusの突然の死でした。

スペック

  • Redmi Note 7 Pro (大陸版)
    • Snapdragon 675
    • メモリ 6GB / ストレージ 128GB
    • 6.3インチ / 解像度 1080x2340
    • 4000mAh
    • Type-C
    • お値段26000円

主な使用目的はブラウジングとたまにゲームと言う感じなのでSoCはあまり重視せずに値段重視。

趣味的な面でROM焼きもするのでSnapdragon搭載機を選びました。

届くまで

注文したサイトはbanggoodで配送業者はAir Parcel Registerでした。注文日から16日で到着。この業者はネットで見てみると地獄の世界一周ツアー(スイス経由らしい)の配送もあったらしいので速い方だったと思います。

使ってみる

Redmi Note 7 Proは現状グローバル版は販売していないようで、中国版かインド版のみ。言語設定は中国語 英語 ヒンディー語といったところでしょうか。自宅に届いてからSIMを刺してすぐにBLU申請をしましょう。約一週間でロックが解除されます。

中国ROMでもGoogle Playを利用する方法はあり、MIUIのままでも問題なく使用することはできます。

私の回線はLINE MOBILE(Docomo回線)でしたが使用するにあたって困ることはありませんでした。APN設定をすれば問題なく使うことができます。(その後Softbank回線に変更して使用中)

スペックも申し分ない感じ。ミリシタもヌルヌル動きます。

デレステはノッチ端末に非対応なので左端の黒い縁が気になる…

カメラもいい感じに撮れるので満足。ってか自分の撮り方が下手くそすぎるのでうまくなりたいですね。(使用アプリはDedicated GCAM for RN7P)

今現在はcrDroid Android 9.0で使用中。Android 10に移るのはもう少し先にしようかと考え中。いい感じのROMを見つけて移りたいなと思ってます。
